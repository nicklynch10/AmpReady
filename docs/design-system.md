# AmpReady Design System

> **Anti-AI Design Philosophy**: Every decision prioritizes human curation, organic imperfection, and tactile warmth over algorithmic perfection. The goal is a website that feels handcrafted by a thoughtful designer, not generated by a machine.

## Texture & Effects

### Grain Overlay

Applied globally at 8% opacity using SVG `feTurbulence`:

```css
.grain-overlay {
  position: fixed;
  top: -50%;
  left: -50%;
  width: 200%;
  height: 200%;
  pointer-events: none;
  z-index: 9999;
  opacity: var(--grain-opacity, 0.08);
  background-image: url("data:image/svg+xml,%3Csvg viewBox='0 0 200 200' xmlns='http://www.w3.org/2000/svg'%3E%3Cfilter id='noise'%3E%3CfeTurbulence type='fractalNoise' baseFrequency='0.85' numOctaves='4' stitchTiles='stitch'/%3E%3C/filter%3E%3Crect width='100%25' height='100%25' filter='url(%23noise)'/%3E%3C/svg%3E");
  background-repeat: repeat;
  background-size: 200px 200px;
}
```

### Organic Blob Shapes

Use irregular border-radius percentages instead of perfect circles:

```css
.blob {
  border-radius: 60% 40% 30% 70% / 60% 30% 70% 40%;
}
```

Variations:
- `60% 40% 30% 70% / 60% 30% 70% 40%` — Default organic
- `40% 60% 70% 30% / 40% 70% 30% 60%` — Alternate organic
- `30% 70% 50% 50% / 50% 50% 70% 30%` — More extreme

### Focus Styles

```css
:focus-visible {
  outline: none;
  box-shadow: 0 0 0 3px hsla(43, 78%, 47%, 0.5);
  border-radius: 4px;
}
```

Always warm amber, never blue.

## Animation System

### Spring Configurations (Motion.dev)

| Name | Stiffness | Damping | Mass | Usage |
|------|-----------|---------|------|-------|
| `microSpring` | 400 | 30 | 1 | Micro-interactions (hover, tap) |
| `feedbackSpring` | 500 | 35 | 1 | Button feedback |
| `layoutSpring` | 300 | 25 | 1 | Layout changes, card hover |
| `pageSpring` | 200 | 20 | 1 | Page transitions |
| `heroSpring` | 150 | 15 | 1.2 | Hero animations (slower, heavier) |
| `bounceSpring` | 250 | 10 | 1 | Bouncy effects |
| `floatSpring` | 50 | 15 | 2 | Floating elements |

### Animation Patterns

**Page Enter:**
```tsx
initial: { opacity: 0, y: 20 }
animate: { opacity: 1, y: 0, transition: pageSpring }
exit: { opacity: 0, y: -10, transition: { duration: 0.2, ease: "easeIn" } }
```

**Staggered Children:**
```tsx
// Container
animate: { transition: { staggerChildren: 0.08, delayChildren: 0.1 } }

// Items
initial: { opacity: 0, y: 15 }
animate: { opacity: 1, y: 0, transition: microSpring }
```

**Scroll-Triggered:**
```tsx
initial: { opacity: 0, y: 30 }
whileInView: { opacity: 1, y: 0, transition: pageSpring }
viewport: { once: true, margin: "-100px" }
```

**Card Hover:**
```tsx
whileHover: { y: -4, rotate: 0.5 }
whileTap: { scale: 0.98 }
transition: layoutSpring
```

**Button Hover:**
```tsx
whileHover: { scale: 1.03 }
whileTap: { scale: 0.97 }
transition: microSpring
```

### Reduced Motion Support

```tsx
const prefersReducedMotion = useReducedMotion();
// Disable or simplify animations when true
```

CSS fallback:
```css
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
    scroll-behavior: auto !important;
  }
}
```

## Component Patterns

### Buttons

**Primary Button:**
- Background: `--color-primary`
- Text: `--color-text-inverse`
- Border-radius: 0.5rem (rounded-lg, not pill)
- Padding: px-6 py-3
- Shadow: `shadow-md`
- Hover: `bg-primary-light`, `scale: 1.03`
- Focus: Warm amber ring

**Secondary Button:**
- Background: transparent
- Border: 2px solid `--color-text-primary`
- Text: `--color-text-primary`
- Hover: `bg-text-primary text-text-inverse`

**Ghost Button:**
- Background: transparent
- Text: `--color-primary`
- Hover: `bg-primary/10`

### Cards

- Background: `--color-surface` or `--color-background-alt`
- Border-radius: 0.75rem (rounded-xl)
- Border: 1px solid subtle warm tint
- Shadow: `shadow-md`
- Slight rotation: 0.3-0.6deg (alternating directions)
- Grain texture overlay at low opacity
- Hover: lift + rotation change

### Forms

- Input background: `--color-surface`
- Border: 1px solid warm gray
- Border-radius: 0.5rem
- Focus: Warm amber border + glow
- Error: Muted red border, not bright red
- Label: Sentient, medium weight, `--color-text-secondary`

### Navigation

- Background: transparent (scrolls with page)
- Links: Sentient, `--color-text-primary`
- Hover: `--color-primary` with spring underline
- Active: `--color-primary` with subtle background
- CTA: Primary button style

## Accessibility

### WCAG 2.1 AA Compliance

| Requirement | Implementation |
|-------------|----------------|
| Color contrast | All text meets 4.5:1 ratio |
| Focus indicators | Warm amber ring, visible on all interactive elements |
| Skip link | "Skip to main content" at top of page |
| Semantic HTML | Proper heading hierarchy, landmarks, roles |
| ARIA | Labels where needed, expanded states on accordions |
| Screen readers | Alt text on images, live regions for dynamic content |
| Keyboard nav | Full tab navigation, Enter/Space activation |
| Reduced motion | Respects `prefers-reduced-motion` |

### Focus Order
1. Skip link
2. Logo/home link
3. Navigation links (left to right)
4. CTA button
5. Main content headings
6. Interactive elements within content
7. Footer links

## Anti-AI Checklist

Before shipping any page or component, verify:

- [ ] No perfect gradients — use grainy gradients or solid colors
- [ ] No symmetrical layouts — break the grid intentionally
- [ ] No generic stock imagery — use real photography or CSS visuals
- [ ] No system fonts — use Zodiak + Sentient
- [ ] No glassmorphism — use tactile textures
- [ ] No tweened animations — use spring physics only
- [ ] No pure white/black — use warm cream and charcoal
- [ ] No perfect circles — use organic blob shapes
- [ ] No over-rounded corners — subtle rounding (0.5rem max)
- [ ] No AI-generated imagery — real photos only
- [ ] Slight rotations on cards (0.5-1°)
- [ ] Intentional asymmetry in layouts
- [ ] Warm, discovered color palette
- [ ] Editorial typography hierarchy
- [ ] Grain texture visible on close inspection
- [ ] Physics-based micro-interactions

## Photography Guidelines

### Sources (Ranked by Quality)

1. **Stocksy** — Photographer-owned cooperative. Authentic, editorial. ~$35/image.
2. **Nappy** — Culturally diverse, free. Real people, zero stereotypes.
3. **Unsplash** — Explicitly bans AI images. Free, high-quality.
4. **Death to Stock** — Cinematic, raw. ~$19/month.

### Rules
- **Never use AI-generated images** for photography of people or real-world scenes
- Use real electrician/worksite photos
- Prefer authentic, slightly imperfect shots over studio-perfect
- Diverse representation (not just white men in hard hats)
- Warm, natural lighting preferred

### FAL API Usage (If Needed)
- Use ONLY for abstract textures, organic shapes, patterns
- NEVER for people, real-world scenes, or photography
- Recommended prompts: "organic texture", "abstract pattern", "grain overlay"

## Responsive Design

### Mobile-First Approach

```css
/* Base styles for mobile */
/* sm: 640px */
/* md: 768px */
/* lg: 1024px */
/* xl: 1280px */
/* 2xl: 1536px */
```

### Key Breakpoints

| Breakpoint | Layout Changes |
|------------|----------------|
| Mobile (< 640px) | Single column, stacked nav, full-width cards |
| Tablet (640-1024px) | 2-column grids, visible nav, medium cards |
| Desktop (1024px+) | Full layout, asymmetric grids, large typography |
| Wide (1280px+) | Maximum container widths, generous spacing |

### Touch Targets
- Minimum 44x44px for all interactive elements
- Adequate spacing between buttons (min 8px)
- Large tap areas for mobile navigation
