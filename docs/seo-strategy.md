## 6. Technical SEO Checklist

### 6.1 Meta Tags (All Pages)

- [ ] **Title Tag**: 50-60 characters, unique per page, primary keyword first
- [ ] **Meta Description**: 150-160 characters, unique per page, include CTA
- [ ] **Meta Robots**: `index, follow` for all public pages; `noindex, follow` for paginated pages, search results, thank-you pages
- [ ] **Meta Viewport**: `width=device-width, initial-scale=1`
- [ ] **Meta Charset**: `UTF-8`
- [ ] **Meta Author**: `AmpReady` (blog posts)
- [ ] **Meta Date Published**: ISO 8601 format (blog posts)
- [ ] **Meta Date Modified**: ISO 8601 format (blog posts)

### 6.2 Open Graph Tags (All Pages)

- [ ] `og:title` - Unique per page, 60-90 characters
- [ ] `og:description` - Unique per page, 150-200 characters
- [ ] `og:image` - 1200x630px, < 8MB, unique per page type
- [ ] `og:image:alt` - Descriptive alt text
- [ ] `og:url` - Canonical URL
- [ ] `og:type` - `website` (pages), `article` (blog posts)

- [ ] `twitter:image:alt` - Same as og:image:alt
- [ ] `twitter:site` - `@AmpReady`

### 6.4 Canonical URLs

- [ ] Every page has a self-referencing canonical tag
- [ ] City pages: canonical includes city slug
- [ ] Blog posts: canonical includes full URL
- [ ] Paginated pages: canonical points to first page with `rel="next"` / `rel="prev"`
